##### Find Pair with give Sum in the Array

```
For example:

Input:
arr = [8, 7, 2, 5, 3, 1]
sum = 10

Output:
Pair found at index 0 and 2 (8+2)
OR
Pair found at index 1 and 4 (7+3)
```
